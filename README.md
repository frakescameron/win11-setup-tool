# Windows 11 Setup Tool

A PowerShell-based Windows 11 post-install automation tool.

This project is designed to help set up a fresh Windows 11 installation by:
- Installing selected applications
- Removing selected preinstalled apps
- Organizing setup tasks into reusable scripts

## Current Status

Early development.

## Planned Features

- App installation using winget
- Bloatware removal
- Interactive setup menu
- Optional Windows tweaks
- Logging
- GUI version in the future
